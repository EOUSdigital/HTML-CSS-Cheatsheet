CSS Selectors

* Specificity list

Value Syntax Description Example
id Selector #idName { }
A hash (#) character followed by the name of the ID. 		Selects elements with a specific ID attribute.
  • HTML: <section id='descriptive'> Descriptive writing helps the reader . . . <section>
  • CSS: #descriptive { font-family: Roboto; }
  • Result: Descriptive writing helps the reader visualize the person, place, thing, or situation being described.
class Selector .className { }
A period (.) character followed by the class name.		 Selects elements with a specific class attribute.
  • HTML: <p class='study'> data science <p>
  • CSS: .study { text-transform: uppercase; }
  • Result: DATA SCIENCE
Type Selector (aka Element Type Selector) element { } Selects elements based on the html element type (e.g. <html>, <head>, <body>, <main>, <footer>).
  • HTML: <h1>This is the text for H1.</h1> <p>This is the text for p.</p>
  • CSS: h1 { color: green; }
  • Result: This is the text for H1.
    This is the text for p.
Universal Selector * { } Selects all elements of any type.
  • HTML: <h1> This is the text for H1.</h1> <p>This is the text for p.</p<
  • CSS: * { color: green; }
  • Result: This is the text for H1. This is the text for p.
Attribute [target] Selects all elements with a target attribute.
  • [attribute*="value"]
  • [class*="te"] { background: color; }
Hover & Cursors :hover To specify the cursor appearance, use the CSS cursor property, which is used to change the mouse cursor type on elements. It can be useful in websites where different actions should be done rather than just clicking.
  • /* Global values */
  • cursor: inherit;
  • cursor: initial;
  • cursor: revert;
  • cursor: revert-layer;
  • cursor: unset;
:nth :nth-child(number) { css declarations; } The selector matches every element that is the nth child of its parent.
  • p:nth-child(odd) { background: oddColor; }
  • p:nth-child(even) { background: evenColor; }

HTML Table Reference

* Table Tags

Content Tag Name Description
# 1 <table> Table The wrapper element for all HTML tables.
# 2 <thead> Table Head The set of rows defining the column headers in a table.
# 3 <tbody> Table Body The set of rows containing actual table data.
# 4 <tr> Table Row The table row container.
# 5 <td> Table Data The standard data cell in an HTML table.
# 6 <tfoot> Table Foot The set of rows defining the footer in a table.

* Table Attributes

Content Attribute Name Description
# 1 colspan Column Span Defines how many columns a td element should span.
# 2 rowspan Row Span Defines how many rows a td element should span.
